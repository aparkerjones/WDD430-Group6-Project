# Theme (Week 02)

## Visual Direction
A clean, optimistic productivity style with strong contrast, warm neutrals, and a high-clarity action color.

## Color Palette
- Background: #F7F5F2
- Surface: #FFFFFF
- Primary text: #1E1F24
- Secondary text: #5A6170
- Accent/action: #0E7490
- Success: #2E7D32
- Warning: #B45309
- Error: #B91C1C
- Border: #D9DDE5

## Typography
- Headings: Poppins, 700
- Body: Source Sans 3, 400-600
- UI labels/buttons: Source Sans 3, 600

## Layout Principles
- Mobile-first responsive layout
- 8px spacing system
- Content width limit for readability
- Primary CTA consistently visible on key screens

## Components to Design First
- App shell (header/nav)
- Auth forms
- Item list/table cards
- Filter/search bar
- Modal or slide-over for create/edit actions
