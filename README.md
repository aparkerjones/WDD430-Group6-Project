## Run the Application
1. Open a terminal in the repository root.
2. Change into the app folder:

```bash
cd handcrafted-haven
```

3. Install dependencies (if needed):

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open http://localhost:3000
