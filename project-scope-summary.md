# W02 Group Project Scope Summary

## Purpose
Our goal for Week 02 is to get the group project organized and ready for development. That means agreeing on the scope, setting up the shared tools, and making the first design decisions together.

## What the Assignment Requires
1. Meet as a group and talk through the project requirements and scope.
2. Create one public GitHub repository for the group project.
3. Set up a GitHub Project board tied to that repository.
4. Invite the rest of the team as collaborators.
5. Add a project README with the group member names.
6. Make sure every member has a local copy of the repository.
7. Define the first pass of the design direction, including the color palette, typography, layout ideas, and early component thinking.
8. Brainstorm user stories and work items, with at least 10 items on the board.
9. Choose the next group leader.
